# Mini Jira

## Overview

Mini Jira is a project management application built using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and JWT Authentication.

## Features

* User Registration
* User Login
* Forgot Password
* Dashboard Analytics
* Project Management
* Task Management
* Project-wise Task Tracking
* Kanban Board
* Calendar View
* Reports Dashboard
* User Profile Management
* JWT Authentication
* Archive / Unarchive Projects

## Technologies Used

Frontend:

* HTML
* CSS
* JavaScript

Backend:

* Node.js
* Express.js

Database:

* MongoDB

Authentication:

* JWT Token

## Run Project

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

Open login.html using Live Server.
